Sorting

array.sort(function(a,b))

If function returns < 0 then a should come first.

If function returns 0 then a and b are equal.

If function returns > 0 then b should come first.

When sorting numbers...

Ascending 1, 2, 3...

function(a,b) { return a - b; }

Descending 3, 2, 1...

function(a,b) { return b - a; }
